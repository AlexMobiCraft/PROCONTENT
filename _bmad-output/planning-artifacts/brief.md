# Product Brief: PROCONTENT

## Executive Summary

PROCONTENT — закрытая платная веб-платформа-сообщество для начинающих
создательниц контента. Подписка 12,99€/месяц или 34€/3 месяца. Ядро продукта —
принадлежность к профессиональному сообществу, а не просто доступ к материалам.
Платформа объединяет регулярный экспертный контент, живое общение и оффлайн-
встречи под руководством автора. Управление подписками через Stripe с
автоматическим отключением. Миграция 2-летнего архива из Telegram через API.

---

## Core Vision

### Problem Statement

Начинающим создательницам контента не хватает профессионального сообщества
единомышленниц с живым ментором, структурированным архивом знаний и реальными
возможностями для роста — всё это в одном месте по доступной цене.

Telegram даёт контент, но не даёт: удобной навигации по архиву, ощущения
профессиональной принадлежности, автоматического управления подписками и
инструментов для масштабирования.

### Problem Impact

- Участницы теряются в 2-летнем архиве и не могут быстро найти нужное
- Ручное управление подписками отнимает время автора
- Нет чёткого onboarding пути — новички не знают с чего начать
- Нет лестницы роста: начинающая → опытная → ментируемая → эксперт
- Платформа не позволяет монетизировать рост участниц (курсы, менторство)

### Why Existing Solutions Fall Short

| Решение        | Недостаток                                                              |
|----------------|-------------------------------------------------------------------------|
| Telegram       | Нет навигации, ручные подписки, нет onboarding, нет ощущения клуба     |
| Patreon/Boosty | Чужой бренд, нет сообщества, нет категоризации, высокая комиссия       |
| Курс-платформы | Статичный контент, нет живого сообщества, нет обновлений               |
| Substack       | Email-формат, нет комментариев, нет категорий, нет чата                |

### Proposed Solution

Веб-платформа с платной подпиской, где:

**Ядро (MVP):**
- Лента постов в стиле Instagram/TikTok — быстрый скролл, визуальные
  карточки с форматом контента, фильтры рубрик прямо над лентой
- Стартовая страница onboarding: знакомство с сообществом, путь для
  новичка, топ-5 постов "начни здесь", ссылка на WhatsApp-чат
- Категории и рубрики: тема месяца, #insight, #monday-inspo и др.
- Комментарии к каждому посту
- Stripe-подписка (12,99€/мес или 34€/3 мес) с автоотключением
- Email-уведомления о новых публикациях
- Превью выбранных автором постов для незарегистрированных
- Mobile-first дизайн
- Миграция архива из Telegram через API

**Масштабирование (v2+):**
- Онлайн-курсы и личные консультации
- Мультиэксперт-каналы (другие эксперты со своими подписками)
- Мобильное приложение
- Менторство как отдельный продукт

### Key Differentiators

1. **Принадлежность, не контент** — закрытый клуб создаёт идентичность и
   удержание сильнее, чем любой отдельный пост
2. **Живой эксперт-автор** с постоянно обновляемым актуальным контентом
3. **Лестница роста** — начинающая участница растёт внутри экосистемы годами
4. **Оффлайн-встречи** — уникальный формат связи онлайн + офлайн
5. **Передача предложений от брендов** — прямая практическая ценность
6. **Доступная цена** при высоком качестве: 12,99€ — осознанный выбор
   без барьера входа
7. **Готовая аудитория** — 10-15 участниц из Telegram готовы к переходу,
   валидация MVP до публичного запуска

## Target Users

### Primary Users

#### Персона 1 — Анна, Начинающая Креаторка (приоритетный сегмент)

**Профиль:**
- Возраст: 22–30 лет
- Ситуация: работает на основной работе или учится, развивает Instagram
  параллельно как дополнительный доход и самовыражение
- Мечта: зарабатывать на съёмке контента для брендов, развиться как
  инфлюенсер или UGC-креатор
- Устройство: почти исключительно смартфон

**Как она переживает проблему:**
- Смотрит на других креаторов в Instagram и не понимает, с чего начать
- Снимает контент, но чувствует, что "что-то не так" — не знает что именно
- Пишет брендам и получает отказы или тишину, не понимает почему
- Чувствует себя одинокой в этом пути — нет окружения, которое её понимает
- Ищет информацию в YouTube и TikTok, но она разрозненная и не системная

**Что она получает в PROCONTENT:**
- Конкретные знания: как снимать, как вести профиль, как работать с брендами
- Уверенность в своих способностях через практику и обратную связь
- Сообщество единомышленниц на том же уровне — поддержка без осуждения
- Новые знакомства на оффлайн-встречах
- Простые объяснения сложных вещей от эксперта

**Реальные отзывы участниц:**
> "Стала реально лучше снимать"
> "Нравится быть частью сообщества и приходить на встречи"
> "Даёт очень много подробной информации и умеет просто объяснять сложные вещи"

**Момент "aha!":**
Когда она отправляет бренду питч по структуре из клуба — и получает
первое "да" на сотрудничество.

**Лестница роста внутри PROCONTENT:**
Новичок → уверенный креатор → первые бренды → менторство →
потенциально эксперт на платформе

### User Journey — Анна (первичный сегмент)

**1. Открытие:**
Видит пост или Reels автора в Instagram → переходит в профиль →
находит ссылку на PROCONTENT → попадает на лендинг со
специально выбранным preview-контентом

**2. Решение о покупке:**
Читает preview-посты (выбранные автором) → видит реальные отзывы
участниц → цена 12,99€ не вызывает сомнений → оплачивает через Stripe

**3. Первый день (onboarding):**
Попадает на стартовую страницу →
"Привет, ты теперь часть PROCONTENT!" →
Читает как устроен клуб →
Открывает топ-5 постов "начни здесь" для новичков →
Переходит по ссылке в WhatsApp-группу →
Знакомится с другими участницами

**4. Ежедневное использование:**
Получает email о новой публикации →
Открывает сайт на телефоне →
Скроллит ленту в стиле Instagram →
Читает/смотрит пост →
Пишет комментарий или применяет совет на практике →
Видит результат → возвращается

**5. Момент ценности:**
Первое успешное сотрудничество с брендом по методике из клуба →
Делится результатом в WhatsApp-группе →
Получает поддержку от участниц →
Продлевает подписку на 3 месяца

**6. Долгосрочное удержание:**
Приходит на оффлайн-встречу →
Знакомится лично с автором и участницами →
Становится "старожилом" клуба →
Советует PROCONTENT подругам →
Рассматривает менторство как следующий шаг

## Success Metrics

### User Success Metrics

Участница получает ценность, если:

- ✅ **Продлила подписку** — главный индикатор того, что клуб работает
- ✅ **Пришла на оффлайн-встречу** — показатель глубокой вовлечённости
  и принадлежности сообществу

**Поведенческие индикаторы вовлечённости (вторичные):**
- Открывает email-уведомления о новых публикациях
- Комментирует посты на сайте
- Активна в WhatsApp-группе

### Business Objectives

| Горизонт       | Цель                          | MRR (при 12,99€/мес)  |
|----------------|-------------------------------|------------------------|
| Запуск (beta)  | 10–15 участниц из Telegram    | ~130–195€/мес          |
| 3 месяца       | 30 платных участниц           | ~390€/мес              |
| 12 месяцев     | 50 платных участниц           | ~650€/мес              |

**Стратегические цели:**
- Автоматизировать управление подписками (0 ручных операций)
- Перенести 100% архива из Telegram без потери контента
- Создать основу для масштабирования: курсы, менторство, мультиэксперт

---

### Key Performance Indicators

**Рост и привлечение:**
- Новых подписчиц в месяц: цель +5–10 начиная с месяца 2
- Конверсия лендинга (посетитель → подписчица): цель ≥ 5%

**Удержание (самый важный блок):**
- Renewal rate месяц к месяцу: цель ≥ 70%
- Доля участниц на 3-месячной подписке: цель ≥ 30%
  *(снижает churn, увеличивает предсказуемость дохода)*

**Вовлечённость:**
- Посещаемость оффлайн-встреч: цель ≥ 40% от активных участниц
- Email open rate: цель ≥ 40%
- Комментарии: цель ≥ 1 комментарий на пост в среднем

**Здоровье платформы:**
- Churn rate в месяц: цель ≤ 15%
- Время до первого входа после оплаты: цель ≤ 24 часа
  *(показатель качества onboarding)*

## MVP Scope

### Core Features (MVP — v1)

#### 1. Контент-платформа
- Лента постов в стиле Instagram/TikTok — карточки с индикатором формата
  (видео, фото, статья, ссылка), быстрый скролл
- Форматы постов: длинная статья с изображениями, короткий текстовый пост,
  только фото/галерея, видео, ссылка на Instagram/Telegram профиль
- Категории и рубрики: тема месяца, #insight, #monday-inspo и пользовательские
- Фильтры рубрик прямо над лентой (не спрятаны в меню)
- Поиск по контенту архива по ключевым словам
- Комментарии к каждому посту (только для подписчиц)

#### 2. Доступ и подписки
- Stripe-интеграция: подписка 12,99€/мес и 34€/3 мес
- Автоматическое отключение доступа при неоплате
- Превью-посты для незарегистрированных — выбираются вручную автором
- Лендинг для незарегистрированных с описанием клуба и отзывами

#### 3. Onboarding и сообщество
- Стартовая страница для новых участниц: знакомство с клубом,
  путь "начни здесь" (топ-5 постов для новичков), ссылка на WhatsApp-группу
- Email-уведомления о новых публикациях (автоматические)

#### 4. Telegram-миграция
- Автоматический импорт архива через Telegram API
- Сохранение дат, форматов и структуры постов
- Категоризация импортированного контента

#### 5. Административная панель (для автора)
- Публикация и редактирование постов всех форматов
- Назначение категорий и рубрик
- Выбор превью-постов для незарегистрированных
- Просмотр списка активных подписчиц и статусов через Stripe
- Управление ссылкой на WhatsApp-группу

#### 6. Технические требования
- Mobile-first дизайн (приоритет — просмотр с телефона)
- Быстрая загрузка медиафайлов (видео, фото)

Переходим к v2 когда:
- ✅ 30+ активных платных участниц

- ✅ 10–15 beta-участниц подтвердили удобство платформы

**v2 — Монетизация роста (6–12 месяцев после запуска):**
- Онлайн-курс как отдельный продукт

- iOS и Android приложение
- Доступ ко всему архиву из веб-версии
- Push-уведомления вместо email
